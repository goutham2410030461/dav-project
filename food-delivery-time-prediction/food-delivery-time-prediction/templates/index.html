Project Overview

This website analyzes and predicts food delivery time using a simple, interpretable model. It also provides clean visualizations to understand how factors like distance, weather, traffic, and order volume affect delivery time.

Dataset

50+ rows with categorical Weather (Sunny, Cloudy, Rainy, Stormy) and Traffic (Low, Normal, Moderate, High).

Model

OneHotEncoder + Linear Regression (scikit-learn). Easy to explain in viva.

Visualizations

Scatter (Distance vs Delivery Time) and bar charts for Weather & Traffic.

Deployment

Flask app, deployable on Render with Gunicorn. Instructions in README.
